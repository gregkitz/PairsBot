## Key Integration Points

1. **Data Processing вҶ’ Pair Selection**: Data processors feed cleaned data to pair selection algorithms
2. **Signal Generation вҶ’ Strategy**: Signals are consumed by strategy implementations
3. **ML Models вҶ’ Signal Enhancement**: ML models enhance signals before strategy consumption
4. **Strategy вҶ’ Execution Environment**: Strategies generate orders executed by the environment
5. **Execution вҶ’ Performance Measurement**: Execution results feed into performance calculation
6. **Configuration вҶ’ All Components**: Configuration system manages parameters for all components
7. **Risk Management вҶ’ Strategy**: Risk management influences position sizing and trade execution

## Notable Cross-Cutting Components

1. **Logging System**: Used across all components
2. **Configuration Management**: Affects all components
3. **Performance Optimization**: Caching and parallel processing applied to various components
4. **Visualization Tools**: Used by multiple components for analysis and reporting
5. **Web Interface**: Provides a unified view and control over multiple components
